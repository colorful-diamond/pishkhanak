رشته ای با طول حداکثر ۴۰ کاراکتر شامل حرف و عدد. پیشنهاد ما برای trackId تولید کردن UUID است. در صورت نیاز به نتیجه یک فراخوانی میتوانید با trackId آن را فیلتر کنید. در گزارش فراخوانی سرویس فراخوانی مورد نظر را با همین کد پیگیری پیدا میکنید. * example: ```block-jh238dfkjbh238sdfge``` * **nid:** (اجباری) &#x202b; کد ملی معتبر که باید به صورت ده رقمی باشد * example: ```0123456789``` ###Results Format #### Successful result format (status code 200) ```````````````` { "trackId": "8edc10e6-1a0b-4f6c-843c-b0397a6c7f9e", "result": { "isBlocked": false, "nid": "0123456789", "message": "کاربری با کدملی موردنظر یافت نشد." }, "status": "DONE" } ```````````````` * **result: ** شرح مسدودی * **isBlocked: ** فلگ مبنی بر بلاک بودن یا نبودن کاربر * **message: ** پیغام یافتن کاربر * **nid: ** کد ملی * **trackId:** کد پیگیری است که با استفاده از آن میتوانید با استفاده از سرویس های فینوتکی کارمزد را استعلام کنید * **status: ** * DONE: فراخوانی موفق سرویس * FAILED: فراخوانی ناموفق سرویس #### UnSuccessful result format در اینجا میتوانید لیست خطاهای سرویس ها را مشاهده کنید [لیست خطاها](/doc/generic-errors.html)
