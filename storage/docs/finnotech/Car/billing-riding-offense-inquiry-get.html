کد پیگیری، رشته ای اختیاری با طول حداکثر ۴۰ کاراکتر شامل حرف و عدد. در صورت ارسال trackId، فراخوانی سرویس خود را با همین مقدار استعلام و پیگیری کنید.(در گزارش فراخوانی سرویس ها با همین رشته نتیجه را ببینید). در صورتیکه که این فیلد را ارسال نکنید یک رشته UUID برای این فراخوانی در نظر گرفته میشود و در پاسخ فراخوانی برگردانده میشود. * example: ```ridingOffenceTrackId``` * **plateNumber:** شماره پلاک موتور * example: ```12345678``` * **nationalID:** کدملی * example: ```0014578963``` * **mobile:** شماره موبایل * example: ```09123456789``` ####Results Format ##### Successful result format (status code 200) ```````````````` { "trackId": "ridingOffenceTrackId", "result": { "Amount": 19395000, "BillID": "8289687000196", "ComplaintCode": "0", "ComplaintStatus": "شکايت ندارد", "PaymentID": "0001996805400", "PlateNumber": "12345678" }, "status": "DONE" } ```````````````` * **result:** آبجکتی از پاسخ سرویس شامل: * **Amount:** مبلغ خلافی * **BillID:** شناسه قبض * **ComplaintCode:** کد شکایت * **ComplaintStatus:** وضعیت شکایت * **PaymentID:** شناسه پرداخت * **PlateNumber:** شماره پلاک موتور * **status: ** وضعیت فراخوانی سرویس * DONE: فراخوانی موفق سرویس * FAILED: فراخوانی ناموفق سرویس * **trackId: ** کد پیگیری، اگر ارسال شده باشد همان مقدار و در غیر اینصورت یک رشته تصادفی تولید و برگردانده میشود * **error:** جزییات خطا (در صورت بروز خطا) #### UnSuccessful result format در اینجا میتوانید لیست خطاهای سرویس ها را مشاهده کنید [لیست خطاها](/doc/generic-errors.html)
