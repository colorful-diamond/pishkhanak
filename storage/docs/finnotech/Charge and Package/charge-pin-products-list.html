کد پیگیری، رشته ای اختیاری با طول حداکثر ۴۰ کاراکتر شامل حرف و عدد. در صورت ارسال trackId، فراخوانی سرویس خود را با همین مقدار استعلام و پیگیری کنید.(در گزارش فراخوانی سرویس ها با همین رشته نتیجه را ببینید). در صورتیکه که این فیلد را ارسال نکنید یک رشته UUID برای این فراخوانی در نظر گرفته میشود و در پاسخ فراخوانی برگردانده میشود. * example: ```getUserinfo781432``` ####Results Format ##### Successful result format (status code 200) ```````````````` { "responseCode": "FN-PGFH-20000700000", "trackId": "iam-track-id-in-uuid-format", "result": [ { "amount": 100000, "productCode": 11, "operator": "MTN" }, { "amount": 50000, "productCode": 10, "operator": "MTN" }, { "amount": 20000, "productCode": 9, "operator": "MTN" }, { "amount": 10000, "productCode": 8, "operator": "MTN" } ], "status": "DONE" } ```````````````` * **result:** آبجکتی از پاسخ سرویس نشان‌دهنده لیستی از محصولات شارژ: * **amount:** قیمت * **productCode:** کد محصول * **operator:** اپراتور #### UnSuccessful result format در اینجا میتوانید لیست خطاهای سرویس ها را مشاهده کنید [لیست خطاها](/doc/generic-errors.html)
